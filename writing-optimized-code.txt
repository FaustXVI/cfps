Writing optimized code

Did you ever had to optimize code ?
Are you sure you really had to do it ?
Did you succeed ?
If you answered "no" to any of this questions, then this talk is for you.
In this language agnostic presentation, you'll learn why, when and how you should optimize your code.
We will answer questions like :
- "How much should I optimize my code ?"
- "How do I optimize without breaking thing ?"
- "Is speed and readability compatible ?"

Duration : 20 minutes
